# COVID-19 Data Analysis and Interactive Dashboard Using Power BI

## 📌 Project Overview
The COVID-19 pandemic generated a massive amount of healthcare data globally. This project focuses on analyzing COVID-19 datasets and creating an interactive Power BI dashboard to visualize confirmed cases, active cases, recoveries, deaths, and country-wise trends.

The dashboard helps users gain meaningful insights into pandemic patterns and supports data-driven decision-making through interactive visualizations.

---

## 📖 Abstract
The COVID-19 pandemic generated a large amount of global healthcare data. This project focuses on analyzing COVID-19 datasets and developing an interactive dashboard using Power BI to visualize confirmed cases, active cases, recoveries, deaths, and country-wise trends.

Multiple datasets such as worldometer data, country-wise statistics, and day-wise reports were integrated and transformed. Visualization techniques were used to understand disease spread and provide useful insights.

**Keywords:** COVID-19, Power BI, Data Analysis, Dashboard, Visualization

## 📈 Dashboard Description

The developed dashboard contains:

### KPI Cards
- Maximum Active Cases: **3M**
- Maximum Deaths: **148K**
- Total Recovery: **9M**

### Visualizations
1. Country Filter (Slicer)
2. Active Cases Indicator
3. Death Analysis Chart
4. Recovery Trend Chart
5. New Cases Analysis
6. Donut Chart for Confirmed, Recovered, and Deaths
7. Country-wise Table Summary

### Key Insights
- Brazil reported very high death counts
- US had the highest confirmed and active cases
- Recovery trends increased across many countries
- New cases fluctuated significantly among regions
